1.怎么判断一个文件是不是一个能被虚拟机接受的Class文件？
  通过Class文件的头4个字节（称为魔数，很多文件都使用魔数进行身份识别，例如GIF或JPEG文件），Class文件的魔数为0XCAFEBABE
  可通过16进制编辑器（UltraEdit）打开.class文件查看

2.类的生命周期
  加载->验证->准备->解析->初始化->使用->卸载
  其中加载、验证、准备、初始化、卸载5个阶段的顺序是确定的。解析可能发生在初始化之后(为了支持动态绑定)

  public static int i = 123;准备阶段值设置为默认值0
  public static final int i = 123准备阶段值设置为123

  3.2 对类进行反射调用时
